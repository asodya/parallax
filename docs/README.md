# Parallax Multi-Screen Quest Plan

Este diretorio registra o plano de implementacao para suportar ate 3 telas do Linux no Meta Quest.

## Como usar

1. Execute as etapas em ordem (`01` -> `07`).
2. Em cada etapa, marque os itens concluidos no checklist.
3. Ao finalizar uma etapa, registre decisoes e riscos no proprio arquivo.
4. Use os criterios de aceite para validar antes de avancar.
